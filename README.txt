RetroPixelS

A experimental 2D-ish? render engine?, focused on single, raw pixel manipulation and retro effects.

Just experimenting, expect very rough edges.

 Low resolution rendering
 Sprite animation system
 Model position control
 Camera perspective
 Camera position control

 FPS camera control with the Pointer Lock API
 Lighting effects
 Outlines
 Particle system
